# DOFEN-Critical-Analysis
Tests of DOFEN on two datasets (Tic Tac Toe and Coffee Transactions). These tests were conducted as part of the Critical Analysis of DOFEN by Laura Essendrop and Sydney Parsons.

In these tests we utilized the DOFEN method and CatBoost on the datasets cited below. 


References:
Aha, D. (1991). Tic-Tac-Toe Endgame [Dataset]. UCI Machine Learning Repository. 
https://archive.ics.uci.edu/dataset/101/tic+tac+toe+endgame

CatBoost. (n.d.). CatBoost: A fast, scalable, high performance gradient boosting on decision trees library 
(Version 8.7k stars) [Computer software]. GitHub. https://github.com/catboost/catboost

Isaienkov, Y. (2025). Coffee Sales [Data set]. Kaggle. https://doi.org/10.34740/KAGGLE/DSV/11159944

Zhou, H., Chen, J., Zhang, Y., Zhao, X., & Wang, J. (2024). DOFEN: Deep oblivious forest ensemble. In 
Advances in Neural Information Processing Systems 37. https://proceedings.neurips.cc/paper_files/paper/2024/file/4ec160e5bf92ce97ad307500e0db9151-Paper-Conference.pdf.
